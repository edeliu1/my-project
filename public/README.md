# My project - Smart City Web Portal – Fushe Kosova

This project is a web-based Smart City platform developed to provide digital
access to municipal services for the citizens of Fushe Kosova.

The system enables users to view public information, services, announcements,
and allows administrators to manage content through an admin dashboard.

## Features
- User authentication (Login / Register)
- Role-based access (Admin / User)
- News and announcements system
- Municipal services presentation
- Contact form
- Responsive and modern user interface

## Technologies Used
- PHP (PDO)
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP (Apache & MySQL)

## Project Structure

## Admin Access
- Admin users can manage news and content through the dashboard
- Access is restricted based on user role
